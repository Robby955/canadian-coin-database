# Changelog

All notable updates to this dataset are documented here.

This changelog tracks significant data updates. Minor corrections (typos, individual field fixes) are applied continuously and not individually listed.

## March 2026 -- Initial Release

- **5,791 coins** across 6 categories: circulation (1,110), collectible (4,174), bullion (215), subscription (157), set (94), roll (41)
- **70 coin series** with year ranges and production status
- Coverage spans 1858 (Province of Canada) through 2026 (current RCM issues)
- Pre-Confederation provincial coinage: Newfoundland, New Brunswick, Nova Scotia, Prince Edward Island
- Complete circulation coinage for all denominations (1c, 5c, 10c, 25c, 50c, $1, $2)
- All major RCM bullion programs (Silver Maple Leaf, Gold Maple Leaf, Platinum, Palladium)
- Full cross-referencing against Coins and Canada, Saskatoon Coin Club, and Charlton Standard Catalogue
- Data available in JSON and CSV formats
